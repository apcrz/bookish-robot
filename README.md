# Bíblia Offline

Aplicativo da Bíblia Sagrada offline em português.

## Sobre o Projeto

Este é um app de Bíblia simples e leve, desenvolvido para leitura offline.  

Contém a tradução **Nova Versão Internacional (NVI)** e partes em **Almeida Atualizada (AA)**.

## Funcionalidades

- Leitura completa da Bíblia
- Navegação por livros e capítulos
- Suporte a múltiplas traduções (em desenvolvimento)
- Leitura offline
- Leve e rápido
